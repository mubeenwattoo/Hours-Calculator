# HourCalc — Free Hours Calculator

Calculate work hours, overtime, and break time instantly.

**Live site:** https://mubeenwattoo.github.io/Hours-Calculator/

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Pushes to `main` automatically build and deploy to the `gh-pages` branch via GitHub Actions.

Ensure GitHub Pages source is set to: **Deploy from branch → `gh-pages` → `/ (root)`**
